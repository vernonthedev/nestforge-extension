Use [NestForge: OpenAPI Docs](command:nestforge.docs) when your local project is already running and you want to open the configured API documentation endpoint.

Use [Open Extension Docs](command:nestforge.onboarding.openDocs) when you want the extension README with command summaries, settings, and usage notes.

A practical rule:

- Choose `OpenAPI Docs` for project-specific API reference.
- Choose `Open Extension Docs` for help using the extension itself.
